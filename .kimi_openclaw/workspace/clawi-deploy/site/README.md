# CLAWI Website — Share Guide

## Quick Start
1. Unzip this folder
2. Open `index.html` in any browser to preview locally

## Share with Your Brother (Get a Live Link)

### Option 1: Netlify Drop (Easiest — No Account Needed)
1. Go to https://app.netlify.com/drop
2. Drag the **unzipped folder** onto the page
3. Done — you get a live URL instantly (e.g., `https://clawi-abc123.netlify.app`)
4. Copy the link and send it to your brother

### Option 2: GitHub Pages (Free, Permanent)
1. Create a GitHub repo at https://github.com/new
2. Upload these files to the repo
3. Go to Settings → Pages → Select "Deploy from a branch" → Select `main` / `root`
4. Your site goes live at `https://yourusername.github.io/clawi`

### Option 3: Vercel
1. Go to https://vercel.com/new
2. Import your GitHub repo (or drag the folder)
3. Instant deployment with a `.vercel.app` URL
